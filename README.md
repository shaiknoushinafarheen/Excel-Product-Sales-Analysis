# Excel Product Sales Analysis

This project analyzes product sales using Microsoft Excel.

## Project Features
- Product sales data analysis
- Pivot Table summary
- Sales chart visualization
- Quantity and revenue comparison

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Data formatting

## Dataset
Products included:
- Laptop
- Monitor
- Keyboard
- Mouse

## Output
A dashboard showing total sales and quantity analysis for each product.
